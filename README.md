# TEVS Abschlussprojekt — Gruppe 7

Abschlussarbeit zur Lehrveranstaltung **Technologien verteilter Systeme (TEVS)**: verteilte Applikation mit mehreren gleichwertigen Backend-Serviceinstanzen, Replikation und fehlertoleranter Nutzung über einen Client.

## Thema

Umsetzung des Standardprojekts **„Verteiltes Commandcenter“**: Clients senden textbasierte Statusmeldungen mit Geodaten an Server-Nodes; Backend speichert, repliziert und hält die Daten über mehrere Nodes konsistent. Details, funktionale und nicht-funktionale Anforderungen stehen in [`ANGABE.md`](ANGABE.md).
